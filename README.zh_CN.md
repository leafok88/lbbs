简介
=================
本软件旨在为一个纯Web访问的BBS [leafok_bbs](https://github.com/leafok88/leafok_bbs) 提供基于Telnet访问的界面。

系统需求
==================
1) GNU C Compiler  
2) PHP ( Version >= 8.2 )  
3) MySQL database ( Version >= 8.4 )
